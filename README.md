# Tasky

Dailly task scheduler
<br>
<br>
Functions
*Main
- Alarm (notification (window + audio))
- Open any file (.txt,.mp4,.exe,script files, anything that your computer recognize) on any OS (Linux,Windows,MAC)
- Open folder (open specific path)
- Open URL (make your default browser navigate to the specified URL)
- Shutdown PC
*Settings
- Keep application on top of all applications (including notification window)
- Automatically remove items when executed if they need to be executed only once
- Show notification window (if unchecked it will notify you only by audio , if it's unchecked and also no valid audi path is present , it won't have any effect when alarm is triggered)
